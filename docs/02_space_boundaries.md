# Space Boundaries

A central V7 principle is the strict separation of spaces.

## Main spaces in reduced reading

- **Runtime** = active technical truth
- **Kanon** = structured reference space
- **Archiv** = trace and historical evidence space
- **Werkbank** = salvage and decomposition space
- **Vorplanung** = bridge-planning and continuation space

## Why this matters

These spaces may influence one another, but they must not silently become the same thing.

This means:

- Werkbank is not runtime
- planning is not runtime
- archive is not current state
- canon is not active state truth
- symbolic expression does not replace technical structure

## Public consequence

This repository shows only the boundary logic, not the full operational depth behind it.
