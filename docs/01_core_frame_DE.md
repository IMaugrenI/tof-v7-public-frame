# Grundrahmen

V7 ist eine kontrollierte Systemform und keine lose Ansammlung von Features.

## Reduzierte öffentliche Lesart

In diesem öffentlichen Rahmen wird V7 über einige stabile Linien gelesen:

- technische Struktur bleibt von Ausdruck getrennt
- Räume bleiben getrennt, statt ineinander zu kollabieren
- Runtime-Wahrheit ist nicht dasselbe wie Archiv, Kanon, Werkbank oder Vorplanung
- Sprachmodell-Ausgabe ist nicht der Wahrheitskern

## Was V7 ist

V7 darf öffentlich gelesen werden als:

- kontrollierte Strukturform
- Systemrichtung mit ausdrücklicher Trennung
- technisches und symbolisches Modell unter klaren Grenzen
- bewusste Weiterführung statt zufälligem Neubau

## Was V7 nicht ist

V7 ist nicht:

- eine gewöhnliche Bot-Oberfläche
- ein einfacher LLM-Wrapper
- eine flache Chat-Architektur
- ein vollständiger öffentlicher Runtime-Stand
- eine Totaloffenlegung interner Mechanik
