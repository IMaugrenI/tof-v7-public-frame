# Öffentliche Kette

Dieses Repository veröffentlicht keine vollständige interne Kette.
Es liefert nur eine reduzierte öffentliche Kettenlesart.

Eine minimale öffentliche Lesart ist:

1. Eingang / Edge-Wahrnehmung
2. Gate- / Governance-Prüfungen
3. Safety / Analyse
4. Worker / State / Retention-Disziplin
5. Archiv / Vector / LLM / Output als spätere Schichten

Der öffentlich wichtige Punkt ist nicht die exakte private Umsetzungsreihenfolge.
Der wichtige Punkt ist, dass Sprachmodell-Ausgabe spät kommt und nicht der Wahrheitskern ist.
