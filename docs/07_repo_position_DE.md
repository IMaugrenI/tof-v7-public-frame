# Repo-Position

`tof-v7-public-frame` soll nicht mit den konkreteren öffentlichen Repos konkurrieren oder sie ersetzen.

Seine Aufgabe ist anders:
- `tof-showcase` = breitere öffentliche Architekturgeschichte
- `tof_local_builder` = lokaler Builder-Stack
- `tof_local_knowledge` = Richtung eines lokalen Wissensprodukts
- `tof-legacy-recovery-workbench` = Legacy-Recovery-Baseline
- `tof-bridge-planning-method` = Planungs-Baseline
- `tof-v7-public-frame` = reduzierter V7-Grenzrahmen

Diese Unterscheidung hält die öffentliche Repo-Familie klarer.
