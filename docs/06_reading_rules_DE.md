# Leseregeln

Dieses Repository soll mit einigen einfachen Regeln gelesen werden:

- öffentliche Rahmung ist keine vollständige Runtime-Offenlegung
- reduzierte Beispiele sind nicht das komplette System
- symbolische oder identitäre Sprache darf technische Wahrheit nicht ersetzen
- Planung, Bergung, Archiv, Kanon und Runtime bleiben verschiedene Räume
- das Fehlen privater Details ist absichtlich, nicht versehentlich
