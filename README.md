# ToF V7 Public Frame

> English is the primary text in this repository. A German clone is available in `README_DE.md`.

Public minimal frame for V7 as a reduced system reading with clear boundaries between runtime, canon, archive, Werkbank, and planning.

This repository does **not** publish the full internal V7 state, its production runtime, or sensitive system internals.  
It presents only a small public-safe frame for how V7 is read as a controlled system form.

## What V7 means here

V7 is not shown here as a full running system.

It is shown only as a reduced public reading of a larger system direction.

In this reading:

- **ToF** is the technical structure and order space
- **V’eth** is the identity, mirror, and expression layer
- V7 is a controlled system form with strict separation of spaces
- public presentation is not the same as full system disclosure

## Purpose of this repository

This repository exists to make visible:

- what V7 is in reduced public form
- what V7 is not
- how major spaces remain separated
- which technical guardrails define the frame
- how a public reading can remain honest without exposing the internal build

## Public boundary

This repository is intentionally limited.

It does **not** expose:

- full runtime structure
- real infrastructure detail
- operational security mechanics
- internal IDs, paths, or secrets
- full service maps
- internal operator logic
- production truth in full depth

This repo is about a **reduced public frame**, not full disclosure.

## Why this matters

Without a reduced public frame, V7 is easily misunderstood in two ways:

- as if it were only a bot or chat setup
- as if public presentation were identical with internal runtime truth

This repository exists to avoid both mistakes while remaining deliberately minimal.
